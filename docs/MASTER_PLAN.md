# Master Plan

This document tracks the Yamato Vikings Hub product plan.

Phase 0 creates the repository, mobile app shell, tooling, CI, deployment configuration, PWA metadata, and documentation structure.

Later phases add the access gate, member profiles, events, RSVP, team generation, voting, fines, MobilePay, seasons, analytics, and exports.

## Phase 0 scope

- React, TypeScript, Vite, Tailwind CSS
- `HashRouter` for GitHub Pages routing
- Mobile portrait shell
- Bottom navigation
- Placeholder Home, Events, Fines, and Members screens
- Supabase local configuration
- Publishable Supabase client setup
- Environment validation
- Vitest, React Testing Library, and Playwright
- ESLint and TypeScript checks
- GitHub Actions CI
- GitHub Pages deployment
- PWA manifest and service worker
- Initial documentation

## Out of scope for Phase 0

- Member registration
- Team-password gate
- Events
- RSVP
- Team generation
- Voting
- Fine Box
- MobilePay payment flow
- Production database schema
